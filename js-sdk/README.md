## React test js-sdk

### Install all dependencies
-----

```
yarn
```
or
```
npm install
```

### Start project on local server (go to localhost:1337)
-----

```
yarn start
```
or
```
npm start
```

### Build project (in "dist" folder):
-----

```
yarn run build
```
or
```
npm run build
```

### Preview minified production-ready bundle (go to localhost:3000)
-----

```
yarn run preview
```
or
```
npm run preview
```
